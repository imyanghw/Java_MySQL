# Java MySQL
MySQL 연동을 통한 간단한 정보 입/출력, 삭제 예시 프로그램

## 개발환경 및 사용버전
- Eclipse IDE(Java)   
- MySQL 5.7
> mysql-connector-java-5.1.45-bin(https://repo1.maven.org/maven2/mysql/mysql-connector-java/5.1.45/)
